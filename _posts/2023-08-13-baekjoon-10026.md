10026번 적록색약 문제는 BFS로 해결할 수 있다.

visited를 생성해 1로 방문 처리를 했다. 그리고 적록색약일 경우 R과 G를 하나의 색(RG)으로 통일시킨 다음 BFS를 적용했다.

답안 코드는 다음과 같다.

```python
from collections import deque

n = int(input())
graph = [[] for _ in range(n)]
for i in range(n):
    data = list(map(str, input()))
    graph[i].extend(data)

# 상, 하, 좌, 우
dx = [0, 0, -1, 1]
dy = [1, -1, 0, 0]


def bfs(x, y):
    queue = deque()
    queue.append((x, y))
    visited[x][y] = 1
    while queue:
        x, y = queue.popleft()
        for i in range(4):
            nx = x + dx[i]
            ny = y + dy[i]

            if nx < 0 or nx >= n or ny < 0 or ny >= n:
                continue
            # 같은 색이면서 방문 안 한 경우
            if graph[nx][ny] == graph[x][y] and not visited[nx][ny]:
                visited[nx][ny] = 1
                queue.append((nx, ny))

# 적록색약이 아닌 경우
visited = [[0] * n for _ in range(n)]
count1 = 0
for i in range(n):
    for j in range(n):
        if not visited[i][j]:
            bfs(i, j)
            count1 += 1

# 적록색약인 경우
for i in range(n):
    for j in range(n):
        if graph[i][j] == 'R' or graph[i][j] == 'G':
            graph[i][j] = 'RG'

visited = [[0] * n for _ in range(n)]
count2 = 0
for i in range(n):
    for j in range(n):
        if not visited[i][j]:
            bfs(i, j)
            count2 += 1

print(count1, count2)
```

예제 입력 1

```
5
RRRBB
GGBBB
BBBRR
BBRRR
RRRRR
```

예제 출력 1

```
4 3
```
